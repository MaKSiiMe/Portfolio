Project Overview

The UNO AI project is a portfolio initiative developed during Holberton School's final sprint. It combines artificial intelligence, game development, and web technologies to create an engaging UNO experience.

Objectives

  • Develop an AI agent capable of playing UNO using reinforcement learning.
  • Create a simplified game engine to simulate UNO matches.
  • Build a web dashboard to visualize game states and AI performance.

Technologies Used

  • Backend: Python, Flask
  • AI: Reinforcement Learning with PyTorch
  • Frontend: HTML, CSS, JavaScript
  • Database: SQLite for storing game data

Project Phases

  1. Stage 1: Team formation and idea brainstorming.
  2. Stage 2: Project charter and scope definition.
  3. Stage 3: Technical planning and architecture design.
  4. Stage 4: Development of the game engine, AI, and web interface.
  5. Stage 5: Testing, documentation, and final presentation.
